## 2. BREND STRATEGİYASI

### 2.1 Brend Pozisiyalaşdırması

```
VAAM = Çin istehsalı ilə qlobal tələb arasında ETİBARLI KÖRPÜ
```

**Unikal Dəyər Təklifi (UVP):**
> "Biz Çinin geniş istehsal bazarında sizin üçün ən yaxşı məhsulları tapır, seçir və təmin edirik — keyfiyyət, şəffaflıq və uzunmüddətli tərəfdaşlıq prinsipi ilə."

- VAAM müştərini mürəkkəb Çin bazarında istiqamətləndirir

## 3. LOGO DİZAYN STRATEGİYASI

### 3.1 Logo Konsept Seçimi

Əvvəlki araşdırmada 6 konsept hazırlanmışdır. Real şirkət profilinə ən uyğun **2 finalçı konsepti** tövsiyə edirik:

#### 🏆 BİRİNCİ SEÇİM: "Golden Compass" (Qızıl Kompas)

**Niyə bu konsept?**
- VAAM-ın əsas funksiyasını — bazarda istiqamətləndirmə və ən yaxşı məhsulu tapmağı — mükəmməl ifadə edir
- Premium korporativ imicə uyğundur (B2B müştərilərin gözləntisi)
- Çox sektorlu fəaliyyəti tək simvolda ifadə edə bilir (kompas = hər istiqamət)
- Bütün ölçülərdə yaxşı oxunur (scalability)
- Mədəni olaraq neytraldır — bütün hədəf bazarlarda (AZ, UZ, RU) eyni mesajı verir
- Zamansız dizayn — trend dəyişikliklərinə davamlıdır

**Logo Elementləri:**
```
┌─────────────────────────────────┐
│                                 │
│         ◇ KOMPAS İŞARƏSİ       │
│      (V-formalı iynə ilə)      │
│     Emerald + Gold gradient     │
│                                 │
│          V A A M                │
│    Xüsusi geometrik font        │
│   (Semibold, 0.12em spacing)    │
│                                 │
│   IMPORT · EXPORT · TRADING     │
│      (Light, 0.4em tracking)    │
│                                 │
└─────────────────────────────────┘
```

**V-formalı kompas iynəsi detayları:**
- Sol qol: Emerald green (#059669) — tapma/axtarış
- Sağ qol: Amber gold (#f59e0b) — dəyər/nəticə  
- Birləşmə nöqtəsi: gradient keçid — transformasiya
- İncə dairəvi çərçivə: slate gray (#94a3b8) hairline stroke
- 4 istiqamət nişanı: N-S-E-W — qlobal əhatə

#### 🥈 İKİNCİ SEÇİM: "Caspian Prism" (Xəzər Prizması)

**Alternativ olaraq niyə saxlanılır:**
- Vizual effekt baxımından ən təsiredici (prizma + spektr rays)
- "Bir ehtiyac — çox həll" mesajını güclü şəkildə ifadə edir
- Glassmorphism trendini 2026-a uyğun istifadə edir
- Premium çap materiallarında çox effektiv (holographic foil)
- 8 fərqli sektorun rənglərlə ifadəsi

**Qərar:** Hər iki konseptin AI alətləri ilə prototipləri hazırlanacaq, müştəriyə təqdim olunacaq, biri seçiləcək.

## ƏLAVƏ C: ACCESSIBILITY VƏ WCAG UYĞUNLUQ

### Rəng Kontrast Nisbətləri (WCAG 2.1 AA)

| Kombinasiya | Kontrast | WCAG AA | WCAG AAA |
|-------------|----------|---------|----------|
| White (#f8fafc) on Navy (#0f172a) | 17.4:1 | ✅ Pass | ✅ Pass |
| Emerald (#059669) on Navy (#0f172a) | 4.8:1 | ✅ Pass | ❌ Fail |
| Emerald (#059669) on White (#f8fafc) | 3.6:1 | ⚠️ Large text only | ❌ Fail |
| Gold (#f59e0b) on Navy (#0f172a) | 8.2:1 | ✅ Pass | ✅ Pass |
| Slate (#64748b) on White (#f8fafc) | 4.6:1 | ✅ Pass | ❌ Fail |
| Slate (#94a3b8) on Navy (#0f172a) | 6.5:1 | ✅ Pass | ✅ Pass |

**Qeyd:** Emerald rəng body text üçün navy fonda istifadə olunmamalı (4.8:1 — large text üçün OK, normal text üçün yetərli deyil). Əvəzinə #10b981 (5.2:1) və ya white istifadə edin.
